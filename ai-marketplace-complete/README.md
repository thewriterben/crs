## 🚀 **Quick Start Guide**

### **Backend Deployment**
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```

### **Frontend Setup**
```bash
cd frontend
npm install
npm run dev
```

### **API Testing**
```bash
curl https://58hpi8c7q968.manus.space/api/ai/dashboard-data
```
