## Understanding Valuations

### Valuation Status

**üü¢ Undervalued**
- Fair value is higher than market price
- Good time to use for payments
- Receive CFV discount

**üü° Fair**
- Fair value matches market price
- Minimal or no discount
- Fair price for both parties

**üî¥ Overvalued**
- Fair value is lower than market price
- No discount applied
- Consider using a different coin

### How Fair Value is Calculated

CFV uses the Digital Gold Standard formula which considers:

1. **Network Security** - Hash rate, node distribution
2. **Utility Value** - Real-world use cases
3. **Scarcity** - Supply metrics and tokenomics
4. **Adoption** - Active addresses, transaction volume
5. **Development** - Code commits, developer activity

This provides a more accurate value than speculation-driven market prices.

## FAQ

### General Questions

**Q: Do I have to use cryptocurrency?**  
A: No, traditional payment methods are also available. CFV discounts only apply to cryptocurrency payments.

**Q: Can I use multiple cryptocurrencies for one order?**  
A: Currently, one cryptocurrency per order. Choose the one with the best discount!

**Q: How often do discounts change?**  
A: CFV calculations update every 5 minutes based on real-time market data.

**Q: Is there a minimum order for CFV discounts?**  
A: No minimum! Discounts apply to all orders paid with supported cryptocurrencies.

### Technical Questions

**Q: What if the CFV calculation seems wrong?**  
A: CFV is calculated using established formulas. If you believe there's an error, contact support with details.

**Q: Can I pay more/less than the exact amount?**  
A: You must send the exact amount shown. Overpayments may not be refunded automatically.

**Q: What happens if I send payment late?**  
A: Payments expire after 15 minutes. You'll need to create a new payment with updated pricing.

**Q: Do network fees affect my discount?**  
A: No, the discount applies to the order total. Network fees are separate and minimal (usually < 0.1%).

### Payment Issues

**Q: I sent payment but it's not confirmed. What now?**  
A: Check the transaction on a blockchain explorer. If confirmed on-chain, contact support with your transaction hash.

**Q: Can I get a refund?**  
A: Refund policies vary by product. Cryptocurrency payments follow the same refund terms as traditional payments.

**Q: What if I send to the wrong address?**  
A: Unfortunately, cryptocurrency transactions are irreversible. Always double-check the address before sending.

**Q: The payment address expired. Do I lose my discount?**  
A: No, create a new payment for the same order. The discount will be recalculated based on current CFV data.

### Security Questions

**Q: Is my cryptocurrency safe?**  
A: Yes! We use industry-standard security practices. Payment addresses are generated uniquely for each transaction.

**Q: Do you store my private keys?**  
A: No! We never ask for or store private keys. You always maintain full control of your cryptocurrency.

**Q: Is my personal information protected?**  
A: Absolutely. We follow strict privacy and data protection regulations.
